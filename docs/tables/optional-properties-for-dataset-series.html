Usage note

description

dct:description

rdfs:Literal

This property contains a free-text account of the Dataset Series. This property can be repeated for parallel language versions of the description.

1..n

first

dct:first

----

This property contains the first resource in an ordered collection or series of resources, to which the current resource belongs.

0..1

accrualPeriodicity

dct:accrualPeriodicity

dct:Frequency

This property contains the frequency at which the Dataset Series is updated. The frequency of a dataset series is not equal to the frequency of the dataset in the collection.

0..1

issued

dct:issued

rdfs:Literal typed as xsd:date or xsd:dateTime

This property contains the date of formal issuance (e.g., publication) of the Dataset Series. The moment when the dataset series was established as a managed resource. This is not equal to the release date of the oldest dataset in the collection of the dataset series.

0..1

last

dct:last

This property contains the last resource in an ordered collection or series.

0..1

modified

dct:modified

rdfs:Literal typed as xsd:date or xsd:dateTime

This property contains the most recent date on which the Dataset Series was changed or modified. This is not equal to the most recent modified dataset in the collection of the dataset series.

0..1

dcat:seriesMember
